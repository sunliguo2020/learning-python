### 知识回顾

## 3 CSS样式

css 专门用来“美化" 标签。

#### 3.1快速了解

```html
<img src=".." style="height:100px"/>

```

#### 3.2 CSS应用方式

##### 1、在标签上

```html
<img src="" style="">	
```



##### 2、在head标签中写style标签

```html
<head>
    <style>
        .c1{
            color:red;
        }
    </style>
</head>
<body>
    
    <h1 class='c1'>
        用户登录
    </h1>
</body>
```

##### 3、写到文件中

```css
.c1{
    height:100px;
}
```



```html
<head>
    
    <link rel='stylesheet' href='common.css'>
</head>
```
